# Simple DeFi Yield Tracker

This project checks DeFi rewards.
It reads data from a blockchain.
You can see how much reward a wallet earned.

This project is made for beginners.
All code is clean and easy.

## Features
- Check staking rewards
- Read smart contract data
- Support for Ethereum testnet
- Simple command line output

## Requirements
- Node.js installed
- Testnet wallet
- RPC provider account

## Setup
1. Clone this repo
2. Open folder in terminal
3. Run: npm install
4. Create a file called .env
5. Add your private key and RPC URL
6. Run: node tracker.js

## Example
Enter wallet address.
The app will show rewards and stake amount.

## Learning Goals
- Learn DeFi basics
- Learn how to read contract data
- Learn blockchain calls
